# Fake mouse


## Installation
You need to install [CircuitPython](https://circuitpython.org/) on your Raspberry Pi Pico. In this project, the Raspberry Pi Pico W was used.
Then download `adafruit_hid` lib and copy it into CIRCUITPY/lib. After that, you cann upload the code from the repository and save it to the device's memory.

## Usage 
Connect the device to the desired PC via USB and enjoy an endless online status.
